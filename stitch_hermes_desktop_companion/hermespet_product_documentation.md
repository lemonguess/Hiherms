# HermesPet 产品设计与功能说明文档

## 1. 产品概述
HermesPet 是一款为桌面端（macOS/Windows）设计的智能助手应用。它以“桌面宠物”为视觉核心，集成了本地/远程 Hermes AI 引擎、语音唤醒交互、Live2D 动态表现以及多源配置管理功能。用户既可以通过传统聊天窗口进行深度交流，也可以通过桌面的悬浮宠物进行轻量级的语音互动。

## 2. 核心界面列表
*   **当前会话 (Main Chat)**: 沉浸式的对话界面，支持上下文追溯、流式文本输出及多媒体操作。
*   **历史记录 (Session History)**: 按时间倒序管理的会话列表，支持搜索和快速回访。
*   **系统设置 (Enhanced Settings)**: 纵向分栏布局，提供 AI 模型参数、TTS 语音引擎 URL、语速音调及系统状态的深度配置。
*   **语音交互反馈 (Pet Overlay)**: 悬浮于桌面的 Live2D 宠物视图，包含实时语音监听字幕气泡和声波反馈状态。

## 3. 功能特性说明
### 3.1 品牌与视觉 (Logo & Identity)
*   **Logo**: 融合科技感与宠物特征的青蓝色矢量图标，代表 HermesPet 的核心身份。
*   **设计语言**: 采用深色模式 (Hermes OS)，利用青蓝色 (#00e5ff) 作为品牌强调色，配合毛玻璃效果 (Backdrop Blur) 提升桌面融合感。

### 3.2 AI 引擎配置 (Brain Configuration)
*   **多源支持**: 支持本地部署的 Hermes 实例及远程 API 接入。
*   **协议兼容**: 深度适配 OpenAI 与 Anthropic 格式的 Base URL 及 API Key。
*   **性能监控**: 右上角状态栏实时显示系统延迟 (Latency) 与内存占用，确保本地模型运行平稳。

### 3.3 语音与交互 (Voice & TTS)
*   **语音唤醒**: 支持 “Hey Hermes” 语音唤醒功能，由顶部声波图标指示监听状态。
*   **实时反馈**: 监听用户输入时，麦克风旁会实时滚动显示识别出的文字。
*   **双行气泡**: 桌宠头部的播报字幕严格限制为两行，确保不遮挡桌面内容，提供简洁的播报反馈。
*   **自定义 TTS**: 允许配置第三方 TTS 引擎 URL，支持音调 (Pitch) 和语速 (Speed) 的微调。

### 3.4 操作交互
*   **消息操作**: 每一条 AI 返回的消息均支持：
    *   **复制**: 一键提取文本。
    *   **播报**: 重新触发语音引擎朗读。
    *   **删除**: 清理当前上下文。
*   **主题切换**: 侧边栏底部提供明暗模式快捷切换。

## 4. 目标平台
*   macOS (Intel/Apple Silicon)
*   Windows 10/11
